# Junior Analyst Vacancy Monitor

Junior Analyst Vacancy Monitor — веб-сервис для анализа рынка труда начинающих специалистов-аналитиков.  
Проект автоматически собирает вакансии с HeadHunter, агрегирует требования работодателей и визуализирует данные по востребованным навыкам и зарплатам, помогая соискателям лучше ориентироваться в ожиданиях рынка.

## Технологии

- **Python 3.10**
- **Django 4.2**
- **SQLite**
- **requests** (интеграция с API HeadHunter)
- **Matplotlib / Plotly** (аналитика и визуализация данных)
- **HTML5, Bootstrap 5**

---

## Скриншоты

### Аналитический дашборд
Графики востребованных навыков и распределения зарплат Junior-аналитиков.


### Список вакансий
Фильтрация вакансий по городу, уровню зарплаты и ключевым навыкам.
